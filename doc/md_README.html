LogServer

This is a python plugin for receiving, storing and displaying the results of the analysis. It was tested:

  • With Python 3.7.1
  • On Ubuntu 18.04

See /doc/index.html for the API.

Dependencies

  • dash, dash_core_components, dash_html_components: Allows easy setup a server with an analytics-oriented visual interface, go to the linked page for intallation instructions
  • plotly: Displays interactive plots of many kinds, go to the linked page for intallation instructions
  • pandas: Among other things, allows easy access to local or online CSV files, go to the linked page for intallation instructions
  • hug: Allows easy setup of basic non-specialized servers, go to the linked page for intallation instructions
